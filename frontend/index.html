Как это работает. Сайт деплоит бота на VDS по SSH и управляет им через HTTP API. После деплоя управляй аккаунтами во вкладке «Аккаунты» — при добавлении выбери VDS. Пароль SSH не сохраняется: при первом деплое ставится deploy-ключ (web/deploy_key). Авто-перезапуск — systemd-таймер на VDS перезапускает бота по расписанию (помогает, если аккаунты «залипают»).
